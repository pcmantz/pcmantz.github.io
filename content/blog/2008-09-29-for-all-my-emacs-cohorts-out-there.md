How do you like having an extremely extensible, mature, and stable text editor. I sure do. One thing that always bugged me when writing code is going through and finding all sorts of annoying trailing whitespace. For most major modes, tabs are converted to spaces for consistency, but trailing whitespaces are neglected. I was considering writing an elisp function to go ahead and take care this until I found [This little gem](http://www.splode.com/~friedman/software/emacs-lisp/src/whitespace.el). I suggest that anyone that has to deal with syntax where trailing whitespace is ignored to have a crack at this.

For that matter, the rest of the interface enhancements on [the main page](http://www.splode.com/~friedman/software/emacs-lisp/) look like they'd be excellent additions to one's `~/elisp/` folder even today; as for the most part, the code here dates back to the early 90's. Emacs isn't exactly a popular mail client anymore...
